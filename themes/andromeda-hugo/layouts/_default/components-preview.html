Image Atom

Advanced image component with Hugo processing, WebP support, responsive srcset, and lazy loading.

Features

  • Automatic WebP conversion with fallback
  • Responsive srcset (4 sizes: 0.5x, 1x, 1.5x, 2x)
  • Lazy loading by default
  • Hugo image processing (resize, fill, fit)
  • Quality optimization (default: 90)

Code Example:

{{`{{ partial "atoms/image.html" (dict
  "src" "images/example.jpg"
  "alt" "Example image"
  "width" 800
  "height" 600
  "responsive" true
  "webp" true
  "lazy" true
) }}`}}
Note: The image atom automatically handles images from assets, page resources, or static directories.

Performance Impact

Button Atom

Code Reduction: ~80 lines → 1 component call

Duplication: 40% → ~10%

Maintenance: Single source of truth

Image Atom

Image Size: 8.3MB → ~2MB (WebP)

Format: Automatic WebP + fallback

Loading: Lazy loading enabled
